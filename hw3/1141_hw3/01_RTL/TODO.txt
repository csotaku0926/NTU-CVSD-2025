Q1: how to do convolution with sram?

8 x sram 512

S = 1, D = 1 : 
06  07 | (00) (01) (02) (03) 04  05 06 07
+++++++++++++++++ 
06  07 | (00) (01) (02) (03) 04  05 06 07

P1
00_P1_r = 00_P3_r + 07 + 00 + 01
..

P2
00_P2_r = 00_P1_r + 07 + 00 + 01

P3
out 00_R0 = 00_P2_r + 07 + 00 + 01 (except for first row) (NOTE: 63, 0 --> 0, 4, set (07+00+01) to 0 instead)
..

----------------

S = 2, D = 1 :
06  07 | (00) X (01) X 02  03
                    06  (02) | (00) (01) (02) (03) 04  05
                                            00  01 (02) 03 (04) 05 06  07
+++++++++++++++++ 


P1 
out 01 = 01_P2_r + 00 + 01 + 02 + 10 + 11 + 12
03_P1_r = 03_P2_r + 02 + 12 + 13

P2


S=1, D=2
00  01 (02) (03) (04) (05) 06  07
-----
00  01   02

P1
out 02' = 02_P3_r + 00 + 02 + 04

S = 2, D = 2
00  01 (02) 03 (04) 05 06  07

P1
out 02' = 02_P3_r + 00 + 02 + 04
out 04' = 04_P3_r + 02 + 04 + 06
